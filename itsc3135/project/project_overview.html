Project Overview

This web application will serve as a reference and analysis tool for Bloons Tower Defense players. It focuses on displaying and comparing all towers and crosspaths that generate money or increase income. Users will be able to view stats, efficiency ratios, and other details through an interactive interface with sortable lists, filters, and visuals :3

  • Purpose: Help players identify the most cost-efficient money towers
  • Intended Users: BTD players who want to maximize their gameplay
  • Content Overview: Tower list, crosspath data, efficiency comparisons, and visual stats

Site Map

The site will include five pages as shown in the sitemap below. Each page connects through a consistent navigation bar for ease of use.

Page Designs

1. Home (index.html)

  • Purpose: Introduce the tool and direct users to tower info
  • Audience: All visitors
  • Content: Hero banner, brief overview, navigation buttons
  • Actions: Navigate to Towers or Efficiency pages

2. Towers (towers.html)

  • Purpose: Display all money-making towers with stats and visuals
  • Audience: BTD players
  • Content: Tower cards with name, image, cost, and income
  • Interactivity: Filter and search bar, sortable lists, expand tower details

3. Crosspaths (crosspaths.html)

  • Purpose: Show how crosspath upgrades affect income
  • Content: Table or grid listing crosspaths and benefits
  • Interactivity: Sort or filter by tower type

4. Efficiency (efficiency.html)

  • Purpose: Compare efficiency ratios of towers
  • Content: Data tables and bar charts of profit per cost
  • Interactivity: Sortable columns, chart display using Chart.js or jQuery plugin

5. About/Contact (about.html)

  • Purpose: Show client/developer info and credits
  • Content: Contact form (mock submission) and site validation links

Dynamic Functionality

  • Filter/search towers by name or type using jQuery
  • Expandable tower cards with toggle details
  • Sortable data tables for cost or efficiency
  • Interactive charts to visualize income efficiency
