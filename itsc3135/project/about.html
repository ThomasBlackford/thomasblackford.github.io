About This Project

This site was created as a simple reference and showcase for BTD6. It’s designed to display all towers and game details in a clean, accessible way.

Development

The project began as a collaboration between friends, with the goal of documenting gameplay and tower stats in one central place. Everything you see here from the layout to the content is built and maintained by a single developer (me :3).

Tools Used

  • HTML, CSS, and JavaScript for structure and styling
  • JSON for storing tower and map data
  • GitHub for version control and hosting
